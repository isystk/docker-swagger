## 📗 プロジェクトの概要

Swagger のサンプルです。Dockerを起動するとSwagger を利用する為の環境が整います。

![Swagger](./swagger.png "Swagger")

![Workflow](./workflow.png "Workflow")

## 💬 使い方

```
# Docker でSwaggerを起動します。
$ docker-compose up -d

# Docker を停止します。
$ docker-compose down
```

Swagger Editor

http://localhost:8001/


Swagger UI

http://localhost:8002/


Swagger API mock

http://localhost:8003/users
